Dieser Button führt auf die Download-Seite von FileZilla. FileZilla wird benötigt um auf den Server zuzugreifen.

Sobald du FileZilla installiert hast must du oben links auf die Serverauswahl gehen und dort einen neuen Server hinzufügen.

Du musst den Servertypen auf SFTP einstellen. Der Host ist "semens.v-server.io" und der Port 22. Der Nutzer ist "root" und das Passwort ist "Semens".

Wenn du den Server hinzugefügt hast kannst du dich über das Servermenü mit ihm verbinden.

Sobald du mit dem Server verbunden bist musst du auf das "/" directory gehen und dann ins "home" directory. Dort ist ein Ordner mit deinem Namen,

Falls dort kein Ordner mit deinem Namen sein sollte kannst du einen neuen erstellen.
